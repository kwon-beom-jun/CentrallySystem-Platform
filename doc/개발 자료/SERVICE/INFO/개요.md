# INFO 서비스 개요

<br>

## 📋 개요

INFO 서비스는 기업 내부의 정보 공유 및 커뮤니케이션을 담당하는 마이크로서비스입니다. 공지사항, 자료실, 자유게시판 등을 제공하여 직원 간 정보 공유와 소통을 원활하게 합니다.

<br>

## 🔧 주요 기능

### 1. 게시판 관리

**역할:** 다양한 유형의 게시판을 관리합니다.

#### 게시판 유형
- **공지사항 (NOTICE)**: 회사 주요 공지사항 게시
- **자료실 (RESOURCE)**: 문서, 자료 공유
- **자유게시판 (COMMUNITY)**: 직원 간 자유로운 소통

#### 게시판 기능
- 게시판 생성, 수정, 삭제
- 게시판 활성화/비활성화
- 게시판별 설명 및 설정 관리

<br>

### 2. 게시글 관리

**역할:** 게시판 내 게시글의 생성, 수정, 조회, 삭제를 관리합니다.

#### 게시글 기능
- 게시글 작성 및 수정
- 게시글 삭제
- 게시글 조회 및 조회수 관리
- 게시글 상단 고정 (공지)
- 게시글 비공개 설정

#### 게시글 권한 관리
- **열람 범위 설정**: 
  - `ALL`: 모든 로그인 사용자 (GUEST 포함)
  - `USER_ABOVE`: USER 이상 권한 (MANAGER, ADMIN 포함)
  - `MANAGER_ABOVE`: MANAGER 이상 권한 (ADMIN만)
- 권한별 게시글 필터링
- 게시글 목록에서 권한 기반 접근 제어

<br>

### 3. 게시글 댓글 관리

**역할:** 게시글에 대한 댓글 기능을 제공합니다.

- 댓글 작성, 수정, 삭제
- 대댓글 지원 (계층형 댓글)
- 댓글 작성자 정보 관리

<br>

### 4. 첨부파일 관리

**역할:** 게시글에 첨부된 파일을 관리합니다.

- 파일 업로드 및 다운로드
- 파일 타입 관리
- 파일 원본명 및 저장명 관리
- 다중 파일 첨부 지원
- 파일 URL 관리

<br>

### 5. 조직 동기화

**역할:** HRM 서비스의 조직 구조 변경 사항을 동기화합니다.

- 부서 생성/변경/삭제 동기화
- 팀 생성/변경/삭제 동기화
- 게시글 가시성 설정에 조직 정보 활용
- 메일 발송 시 조직별 타겟팅

<br>

### 6. 메일 알림 기능

**역할:** 게시글 작성 시 사원들에게 알림 메일을 발송합니다.

- 게시글 작성 시 메일 발송 요청 이벤트 발행
- HRM 서비스로 메일 발송 요청 전달
- 타겟 설정:
  - 전체 사원 발송
  - 부서별 발송
  - 팀별 발송
  - 권한별 발송 (GUEST, USER, MANAGER, ADMIN)

<br>

## 🔄 Kafka 이벤트 통신

### 수신 이벤트 (Consumer)

#### HRM 서비스 이벤트
- **부서 생성/변경/삭제**: 조직 구조 변경 시 INFO 서비스의 부서 정보 동기화
- **팀 생성/변경/삭제**: 팀 구조 변경 시 INFO 서비스의 팀 정보 동기화

<br>

### 발행 이벤트 (Producer)

#### 메일 발송 이벤트
- **게시글 메일 발송 요청**: 게시글 작성 시 HRM 서비스로 메일 발송 요청 전달
  - 게시글 정보
  - 발송 대상 (전체/부서/팀/권한별)
  - 메일 내용

<br>

## 📌 주요 특징

### 1. 권한 기반 접근 제어
- 게시글 열람 범위를 권한 단계별로 제어
- 부서/팀별 게시글 접근 제어 지원
- 비공개 게시글 기능

### 2. 조직 구조 실시간 동기화
- HRM 서비스의 조직 변경 시 자동 동기화
- 게시글 가시성 설정 및 메일 발송에 조직 정보 활용

### 3. 게시글 상단 고정
- 중요 공지사항을 목록 상단에 고정
- 고정 게시글 우선 정렬

### 4. 조회수 관리
- 게시글 조회 시 자동으로 조회수 증가
- 조회 통계 관리

### 5. 다중 첨부파일 지원
- 한 게시글에 여러 파일 첨부 가능
- 파일 타입별 관리

<br>

## 🔗 관련 서비스와의 연동

### HRM 서비스
- 조직 구조(부서/팀) 변경 사항 실시간 동기화
- 게시글 메일 발송 시 사원 정보 조회
- 부서/팀별 메일 발송 타겟팅

### Receipt 서비스
- 공지사항을 통한 영수증 제출 안내
- 자료실을 통한 영수증 양식 제공

<br>

## 💡 활용 사례

- **공지사항 게시**: 회사 주요 공지, 정책 변경 사항 공유
- **자료실 운영**: 업무 양식, 매뉴얼, 참고 자료 공유
- **자유게시판**: 직원 간 자유로운 의견 교환 및 소통
- **부서별 공지**: 특정 부서/팀만 볼 수 있는 게시글 작성
- **조직 변경 알림**: 조직 구조 변경 시 관련 공지 자동 배포
- **파일 공유**: 프로젝트 자료, 회의록, 보고서 등 파일 공유

<br>
